The following functions are available globally.

  • postNotification(_:value:center:)

    Posts the given notification to the specified center. This function has the same type parameters as Notification, namely <V, S: AnyObject>, which restricts the type of value that can be posted.

    Declaration

    Swift

    public func postNotification<V, S: AnyObject> (notification: Notification<V, S>, #value: V, center: NSNotificationCenter = NSNotificationCenter.defaultCenter())

    Parameters

    notification

    The notification to post.
    value

    The data value to be sent with the notification.
    center

    The notification center from which the notification should be dispatched. The default parameter value is NSNotificationCenter.defaultCenter().
